# 📘 Database Management System Lab (DBMS Lab)

## 🧑‍💻 Overview
This repository contains all the practical experiments performed as part of the **Database Management System Lab** course.  
The goal of this lab is to understand how databases are designed, created, manipulated, and managed using SQL and PL/SQL.

---

## 🎯 Objectives
- Understand SQL fundamentals (DDL, DML, DCL)
- Design relational databases using ER diagrams
- Implement database schemas using SQL
- Perform data manipulation operations
- Learn PL/SQL programming (procedures, functions, triggers)
- Handle real-world database problems

---

## 🧱 Technologies Used
- SQL (Structured Query Language)
- PL/SQL (Procedural SQL - Oracle)
- MySQL / Oracle SQL Developer / SQL*Plus

---

## 📂 Experiments Overview

### 🔹 Experiment 1: University Database Design
- Designed a complete university database system
- Created tables like UNIVERSITY, DEPARTMENT, STUDENT, INSTRUCTOR, COURSE
- Implemented relationships using primary and foreign keys
- Focus: ER Diagram + Schema Design

---

### 🔹 Experiment 2: Data Manipulation & Schema Changes
- Inserted data into all tables
- Performed UPDATE, DELETE operations

- Created and used VIEWS

- Focus: DML + Advanced SQL operations

### 🔹 Experiment 3: SQL Functions

### 🔹 Experiment 5: SQL Joins

## 🚀 How to Run
1. Install MySQL / Oracle SQL Developer / SQL*Plus
2. Open your SQL tool
3. Run the SQL scripts experiment-wise
4. Enable output (for PL/SQL):
   ```sql
   SET SERVEROUTPUT ON;
   ```

---

## 📌 Notes
- Make sure tables are created before inserting data
- Follow the correct order to avoid foreign key errors
- Use COMMIT if required
- Always check constraints before deleting data

---

## 💬 Viva Preparation Tips
- Understand difference between PRIMARY KEY & FOREIGN KEY
- Know types of JOINS
- Be clear about TRIGGERS (row vs statement)
- Practice writing queries manually
- Understand real-world use cases
